Web Storage Demo

This is an exercise inspired by this excellent MDN tutorial. It demonstrates the W3C Web Storage API.

Web storage was introduced as an alternative to HTTP cookies. Like cookies, it offers a way for a server to keep track of who it is talking to on the client side.

Web storage comes in two flavors localStorage which persists beyond the current session and sessionStorage which associates multiple windows to a single session. For a discussion see the readme.md

In this demo your color choice will be stored to localStorage. If you close the window and come back, your choice will persist. Your choice of font on the other hand is stored to sessionStorage. This will reset to the default ()
